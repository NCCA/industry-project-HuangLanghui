# Data folder

This folder is intentionally empty in the submitted repository.

Download the real NYUv2 labeled dataset from the official page:
https://cs.nyu.edu/~fergus/datasets/nyu_depth_v2.html

Expected file:

```text
data/nyu_depth_v2_labeled.mat
```

The code will fail with a clear error if this real dataset file is missing. It does not generate dummy depth maps or synthetic replacement data.
